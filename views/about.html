About the Author

My name is Grant Mercer. I'm not any notable slackliner or anything, but I do really enjoy slacklining as well as programming. Slackline Picker is built in Angular.js from scratch, no predefined site templates or code were used (beyond the libraries!). The site is techniacally memoryless, it does not contain any sort of database and instead uses a couple different nead cloud tricks to store and load information.

Building SlacklinePicker was a lot of fun, and borderline was an obession during the time I was building it. I'm about a month or so into learning web development, coming from a background of growing up with C++ as my main language. I hope this site can be of some use to the community, afterall that was my intended goal.

About the Database Used

The database was built based off the Slackchat Brands Page. You'll notice some of these items (or brands) are missing from the database. There are a couple reasons why I was unable to add items/brands:

  • Site was not translatable to English
  • Item is a trickline component(sorry, not supporting these items!)
  • Not related to slack kits (swag)
